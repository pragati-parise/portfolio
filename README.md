# Pragati Portfolio

A React portfolio project built with Vite and ready for GitHub + Netlify deployment.

## Tech Stack

- React.js
- Vite
- CSS

## Project Structure

- `src/App.jsx` - main React portfolio component
- `src/main.jsx` - React entry point
- `src/styles.css` - all styling and responsive layout rules
- `public/` - profile photo and skill logos

## Run Locally

1. Install dependencies with `npm install`
2. Start the dev server with `npm run dev`
3. Build for production with `npm run build`
